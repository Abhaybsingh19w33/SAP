After the MODIF ID addition, 
a modid identifier can be specified to 
assign a screen element of a selection screen 
to a modification group.


What is Modification Group ?

Grouping of multiple screen elements of a 
dynpro under a three-character abbreviation, 
so that the display properties of the dynpro(screen) 
can be modified together using the statement 
MODIFY SCREEN. 

Screen elements are assigned 
to modification groups in Screen Painter 
or in selection screens in the ABAP program 
using the addition MODIF ID.
