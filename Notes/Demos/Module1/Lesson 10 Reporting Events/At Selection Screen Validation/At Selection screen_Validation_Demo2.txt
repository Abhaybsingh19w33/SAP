REPORT z.
TABLES SCARR.

TYPES: BEGIN OF SCARRTYPE,
          CARRID TYPE  SCARR-CARRID ,
          CARRNAME TYPE SCARR-CARRNAME ,
          CURRCODE TYPE SCARR-CURRCODE,
          URL     TYPE SCARR-URL,
       END OF SCARRTYPE.
 DATA ITSCARR TYPE STANDARD TABLE OF SCARRTYPE INITIAL SIZE 0.
 DATA WA TYPE  SCARRTYPE.      .

SELECTION-SCREEN BEGIN OF BLOCK BLK1 WITH FRAME TITLE TEXT-100.
  PARAMETERS: P_CARRID TYPE SCARR-CARRID.
SELECTION-SCREEN END OF BLOCK BLK1.

AT SELECTION-SCREEN ON P_CARRID.
IF P_CARRID IS INITIAL.
  MESSAGE E001(Z104329MSGCLASS) WITH 'CARRID should not be blank'.
ENDIF.
IF P_CARRID NE SPACE.
  SELECT CARRID  CARRNAME CURRCODE URL  FROM SCARR
    INTO TABLE ITSCARR WHERE CARRID EQ P_CARRID.
IF SY-SUBRC NE 0.
    MESSAGE E001(Z104329MSGCLASS) WITH 'NO RECORD PRESENT'.
ENDIF.
ENDIF.

START-OF-SELECTION.
LOOP AT ITSCARR INTO WA.
  WRITE: /  WA-CARRID,WA-CARRNAME ,WA-CURRCODE ,WA-URL.
ENDLOOP.
