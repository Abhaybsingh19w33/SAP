*NO-ZERO COMMAND SUPPRESSES ZEROS LEADING ZEROS FROM THE NUMBER FIELD.
*NO-ZERO ONLY SUPPRESSES THE ZEROS WHILE DISPLAYING.
*NO-ZERO APPLIES ONLY TO NUMERIC FIELDS.

DATA W_N1(10) TYPE N VALUE 70.
WRITE: 'Displaying numeric data without NO-ZERO:', W_N1.
SKIP 1.
WRITE: 'Displaying numeric data with NO-ZERO:', W_N1 NO-ZERO.
