

	Transient data :- data available only till runtime
			eg:- variables declared inside blocks (subroutines,f.m's,methods),
			 global variable declared in a program,
			class objects,

	persistent data:- data available beyond runtime

			eg:- data stored in db tables,
		data stored in 	presentation/appl.server files
